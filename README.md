# Vivento - Digital Invitation Platform

Modern digital invitation and event management platform.

## Tech Stack

- **Frontend:** React + Tailwind CSS
- **Backend:** FastAPI + Python
- **Database:** MongoDB

## Features

- Template-based invitation creation
- Custom template editor
- Event management
- Guest RSVP tracking
- Admin panel
- Custom fonts support
- Google OAuth integration

## Environment Variables

Required environment variables are configured in deployment platforms.

See deployment documentation for setup instructions.
